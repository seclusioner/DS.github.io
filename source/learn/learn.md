# 前言

這篇主要是想記錄一些我對學習的一些想法(**當然會帶有主觀角度，但希望能引起你的共鳴**)，以及學習的資源與過程。<br/><br/>

#  理念

{% note [class] [fa-solid fa-book fa-beat] [style] %}
教育是每個人必經的階段，不論是為了升學成績，還是學習，歸咎於現實面沒有對與錯，但對我而言，與其為了考好學校補習、刷題，一整天瘋狂練習解題技巧，做對學習一點幫助都沒有的事情，不如將理論讀好，從實作過程中累積經驗學習，培養個人各方面的能力，個人能力跟學校一點關係都沒有，所以<strong>我個人絕對不會補習</strong>，雖然好學校資源、環境都比較好，但有能力的人絕對只缺機遇，與其透過各種手段達到目標，我個人喜歡直接挑戰現實，你說我在逃避現實不肯面對？沒關係，我會證明給你們看，什麼都以現實做為考量只會礙手礙腳，有什麼好怕的？做就對了，用行動證明自己，讓那些懷疑你的人只剩心服口服。

小時候讀書我就不是以成績升學為主要的學習的目的，很多人愛比較成績，即使是升學那些也都要成績，甚至為了成績或學分選較少的課程，這些可笑的事在我這邊不會發生，因為我是為了**學習**才來這裡的，如果只是拿了文憑，那你來學校也沒什麼用，雖然抱怨也無濟於事，但我一直想證明：**成績絕對不代表一個人的能力，學東西絕對不是為了考試，而是可以真的把東西學進去，應用在將來，與其一直刷題考上好學校，倒不如把理論、實作融會貫通，培養實力，誰說學校不好能力就不好? 題目只是讓你確認哪裡不會、理解錯誤等，是學習的手段而不是目的。** <br/>

真正學好一樣東西是很不容易的，但在台灣教育之下要真的學自己想學的東西是有點難度的，**制度成為了我學習最大的絆腳石**，當然也可能是我太笨，反正就好好學吧。當然現實是很結果論的，人家也不認識你，第一次當然是看成績，諷刺的是，我雖然有興趣，但還沒有專精的項目，況且心理我想應該還尚未準備好，我在生活上已經算是機器人了，希望能突破自己，我從小就想不透過成績來證明自己的實力，證明給那些只在乎成績的人看，打他們幾巴掌，但我到現在似乎還是沒做到這點，現在是一個重要的轉折點，在這個階段，我需要思考並找到自己需要做些什麼才行。 --- 於某暑假的紀錄

#  學習 & 過程

## 方式 & 過程
建議上OCW之前，可以查看看該課程教授的個人網頁，有些很佛心的會把講義(ppt、pdf)、作業、小考&考試放在網路上，是很好的資源，也可以看看教授的網頁內容及其領域和論文。<br>

上OCW很常轉1.75倍速上下聽，有時候比較難的會慢下來，有些會筆記，有些只看講義之類的，但不複習一定會忘記！！因此搭配課本以及多加練習效果會更好(如有前面說的資源更好)。<br>

## 教育至始至終的問題

像我這篇或其他篇關於"學習"或是"教育"寫的，真正的學習絕對不是考試考高分、考前幾名，學習本身是一件很不容易的事情，意外看到[這篇(作者：NTHUOCW)](https://www.bilibili.com/read/cv13717469/)趙啟超教授的文章，小弟我上過趙教授的線性代數、離散數學的OCW，上的非常好，收穫良多，~~雖然我數學還是不太好~~，裡面有一段讓我印象很深刻：*從小受台灣教育體制影響，習慣所謂「學習」正是學習「如何考試」的模式，「在美國念書的那幾年我覺得對我的影響很大，因為你要知道，你一定要努力才行。這個很多努力是要 **『survive』** 的努力，每個人都不見得不比你聰明，不努力的話你就被淘汰了。」*，即使知道考試不是目的，但你不透過這種方法，也沒辦法上頂大，聽起來格外諷刺，但真正的學習就像這句話說的，是為了**生存**，既現實卻又很符合我的狀況，每天都抱著必死的決心在學東西(其他我所經歷的以及我在這篇沒細說的內容，如果想更瞭解可以去看我其他篇文章)。<br>

# 自學

我想自學可以學到更多東西外，也可以訓練很多能力，但最大的缺點就是耗時，也沒人帶你，需要自己摸索與探索，不斷看書爬文找資料，有時會比較迷茫一點，有 OCW 這類免費資源，也有很多其他的線上課程，我目前是還沒上需要花錢的。<br>

另外希望有機會與有能力的話可以跟不同領域厲害的人交流，不希望僅僅是死讀書而已。<br>

# OCW

## 優點/缺點
OCW是很好的資源，上課的過程中可以知道一些額外的內容與訊息，而且通常頂大的教授上課不論是表達能力、連結、引導方式，於一定的數學基礎下將該科理論與應用結合，有時候會補充一些產業訊息，難怪學生的理論扎實程度與我們這些不知道在幹嘛的學校簡直是天壤之別，且皆與一般的教授(像我們學校)差很多，~~我們教授都在比爛的~~(當然好的也是有好幾個)，不過當然數學或理論的部分學習與否和深入程度因人而異，每個人重視的點和學習方法並不同，但我自己在上OCW時的感受與我們學校上課差很多，上OCW是一種享受，不過相對的上的越多越知道自己不足的地方在哪裡，~~電機真的是一條不歸路~~，優點是就理論的部分確實可以更扎實，而且可以隨時看，沒有時間限制，但缺點也是有，遇到不懂的沒辦法當下問授課教授，且沒有經過安排的考試、作業等練習的過程，一切還得看你個人能力以及如何安排你的學習，OCW是很好的輔助，但也要看課本以及實體授課，才能學得更扎實，以上是我個人觀點。
